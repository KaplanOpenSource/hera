getLandCover(minlon, minlat, maxlon, maxlat, dxdy=30, inputCRS=4326, dataSourceName=None)[source]

Get Xarray LandCover map.

Parameters:

  • minlon (float) – Minimum value of longitude coodinates.

  • minlat (float) – Minimum value of latitute coodinates.

  • maxlon (float) – Maximum value of longitude coodinates.

  • maxlat (float) – Maximum value of latitute coodinates.

  • dxdy (int, default=30) – Spatial resolution of the output land cover map. Defines the step size for the grid points in both the x (longitude) and y (latitude) directions within the specified bounding box. Smaller values result in finer resolution.

  • inputCRS (int , default=WSG84) – The ESPRG of the coordinates.

  • dataSourceName (string , default=None) – The name of the data source to use.

Return type:

xarray.DataArray
